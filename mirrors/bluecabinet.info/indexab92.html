Products

A10 Networks delivers high-performance network management tools that Accelerate, Optimize, Secure, and provide unprecedented visibility for network traffic as it traverses down network pipes.

  • ACOS Advantage

Advanced Core Operating System (ACOS) was designed with the ability to swiftly port to other processors. This makes ACOS a suitable OS for hypervisors, enabling support without substantial re-engineering and compatibility issues. The only full-featured 64-bit ADC virtual appliance on the market with all feature licenses included. As ACOS was also designed to reduce internal overhead and resource usage on the host VM, host hardware is optimized. Ability to run on a single CPU for maximum density. Efficient operation, customized to utilize only the resources required from the hypervisor, eliminating continual polling.


  • ID Series - Network Identity Management

The ID Series contains the industry's only network identity management appliances delivering integrated features that resolve IP addresses to user identity (IP-to-ID) instantly, simplify user password management with the User Self Help Web Portal, provision user account management, and enhance RADIUS authentication and access control. A10 Networks markets ID Series through resellers under the A10 Networks brand, and through agreements with technical partners such as Lancope, RSA (a division of EMC), WildPackets, RetiCorp and FireScope.


  • AX Series - Advanced Traffic Management

The AX Series is a new generation server load balancer, helping enterprises and ISPs to scale and maximize application availability through the world's most advanced application delivery platform. The AX Series is revolutionary by current market standards due to its Advanced Core Operating System (ACOS) with a native multi-core CPU, shared memory architecture, which is exponentially faster and more scalable than other systems on the market. With the AX Series, customers do not have to sacrifice features for performance. Plus, all features are included in the standard list price.


  • EX Series - Bandwidth Management with Identity Reporting

A10 Networks' EX Series is the industry's only bandwidth management appliance with identity-based application visibility. By managing bandwidth and traffic with unprecedented user and application visibility...
