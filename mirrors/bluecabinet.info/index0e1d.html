Background

TamoSoft is a privately held company, built on the capital brought in by the founders.

TamoSoft develops cutting-edge security and network monitoring software for the Internet and Local Area Networks... offer professional tools that support the latest standards, protocols, software, and hardware for both wired and wireless networks.

Products

Commview and Commview for Wifi, packet capture programs similar to Wireshark

  • These support the recording of SIP and H.323 conversations

NetResident, a Deep Packet Inspection (DPI) program

  • This is apparently meant for IT departments, and doesn't look like it'd work on a large scale. It can apparently be used to spy on individuals, though

Smart Whois and CountryWhois, programs to look up information about IP addresses Look-up IP Data Mining

Essential NetTools, a program combining small networking utilities

CommTraffic, a tool to manage expenses for someone who has a cap on their network usage

TamoSoft Throughput Test, a tool to test the throughput of a network connection
