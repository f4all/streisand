IRC unter Linux Installieren und Konfiguieren

Linux

Dieser Abschnitt erklärt die einzelnen Schritte um euch mit dem Anonops IRC Netzwerk via Linux zu verbinden.

GNU/Linux Debian (Ubuntu + SUSE)

Die aktuellste Xchat Version ist unter folgender Quelle zu finden 
Installiert das Xchat Paket

sudo apt-get install xchat
Einfacher geht es kaum.

GNU/Linux Redhat/Fedora

Die aktuellste Version von Xchat befindet sich im EPEL Verzeichnis. Dies ist die einfachste Methode.
Ladet das EPEL Paket über Fedora Projekt

Als nächstes yum install xchat

GNU/Linux Gentoo

Ports Aktualisieren: 
sudo emerge --sync
Und Installieren über: 
sudo emerge -av xchat
Fertig Installiert, einfach Xchat über die Apps-> Internet oder "xchat" über die Konsoleneingabe starten.

XChat Konfiguieren um auf das AnonOps Netzwerk zu gelangen:

1) Klickt den Add Button neben Netzwerk. Links davon steht ein untermalter Text mit New Network, ändert dies in was auch immer ihr wollt. Idealerweise Anonops, um einfacher den Server zu finden.
2) Nun Klickt auf Edit, und ein neues Fenster mit Server für Anonops öffnet sich. 
Wieder auf Edit Klicken und den Text in: irc.anonops.com/6697 ändern.
Haken in das Kästchen: "Use SSL for all the servers on this network"
genauso wie bei: "Accept invalid SSL certificate"
Wenn ihr dies möchtet könnt ihr euch auch Automatisch zum AnonNetzwerk verbinden lassen beim Programmstart. Dazu einfach ein Haken in "Connect to this network automatically". Ist nur Optional, kann aber dem ein oder anderen vielleicht Helfen.
3) Nun den Close Button drücken und ihr werdet wieder zurück zum Netzwerk Fenster gebracht.
Unter Networks, drückt auf euren gewählten Netzwerk Name (für viele wird es Anonops sein) und einfach auf Connect unten rechts Klicken.
Der Client wird sich nun (hoffentlich Erfolgreich) mit dem Netzwerk verbinden.
Wird ein error angezeigt, Überprüft ob ihr "irc.anonops.com" richtig geschrieben habt. Sollte wirklich keine Verbindung mit irc.anonops.com zustande kommen, verwendet irc.anonops.net.
