Premièrement: Quelques définitions

Un “NickName” est votre surnom/pseudonyme sur IRC. Choisissez-en un et gardez-le. Ce nom apparaitra devant toutes les lignes que vous allez écrire . Anonymous signifie anonyme, donc ne réutilisez pas le même surnom sur un autre irc, forums, réseaux sociaux, etc., et n'indiquer aucune information en lien avec votre vie personnelle. Vous pouvez choisir n'importe quel Nick, sauf s'il y a une personne qui l'a choisi avant vous.

Ne pas choisir un Nick du genre 'justinbieber' ou 'Ims0l33t4ndc00l', vous serez probablement humilité, et fouetté, ceylavie bébé ^^ La plupart des nouveaux usagers se connectent via le WebChat, qui s'utilise avec un Navigateur Web (Chrome, IE, Firefox etc..) . AnonOps propose un Webchat SSL chiffré afin que vous puissiez vous connecter. Le webchat étant un intermédiare de plus entre vous et le serveur AnonOps, et ne pouvant pas être configuré, faites en sorte de prendre au plus vite un client irc. (xchat, kvirc, etc.) Utiliser le WebChat pour nous joindre. Entrez sur la page webchat lorsqu’il vous le sera demandé, entrez votre Nick et joignez le Canal #OpNewblood.FR. En étant sur ce canal, vous pourrez nous posez vos questions.

Nous vous invitons d'ailleurs fortement à lire ceci avant de vous lancer dans des questions qui pourraient être inutile ou mal posées : http://www.gnurou.org/writing/smartquestionsfr

Info supplémentaire :

Un « Highlight » c’est lorsque un autre utilisateur écrit votre Nick dans une phrase. Cette ligne apparaitra en rouge. Il est possible que votre clients IRC fasse clignoter une fenetre ou fasse un son pour vous avertir du Highlight.

N'en abusez pas !

Deuxième points: Quelques commandes de base

Une fois connecté à un serveur IRC, vous pouvez voir un message indiquant que vous utilisez un Nick appartement à un autre usager, vous devez donc en changer, en faisant ce qui suit: Pour choisir et/ou changer votre Nick, tapez la commande suivante:

/nick [Votre Nick]
Exemple : /Nick Newblood

*** Vous ne devez JAMAIS mettre les [guillemets ] ***

L’enregistrement de votre Nick.

Une fois votre Nick choisi, enregistrez-le. C'est important car cela permet :

  • Que personne ne le prenne pour se faire passer pour vous;
  • D'enregistrer des Canaux à votre nom;
  • D'être Opérateur sur des Canaux;
  • De recevoir des Mémos;
  • Et bien plus...
Pour enregistrer votre nick, tapez:
/msg nickserv register [VotreMotDePasse] [Un Faux Courriel]
Exemple: /msg nickserv register MotDePasse1 [email protected]
*** Ne perdez pas votre mot de passe, car vous ne pourrez plus le retrouver.***

Naviguer dans IRC

L'identification

A chaque fois que vous venez sur IRC, vous devrez vous identifier, si vous avez enregistré votre Nick auparavant bien sûr.

Pour s’identifier :
/msg nickserv identify [Votre Mot De Passe]
ou
/identify MonMotDePasse

Si vous n’êtes pas capable de vous identifier correctement, votre nick sera changé pour un autre au hasard.

Exemple : Guest1234

Dans ce cas, CHANGEZ votre Nick, et IDENTIFIEZ-VOUS encore (il se peut que vous deviez attendre quelques secondes/minutes avant de pouvoir récupérer votre nick).

Quelques Commandes IRC

*** ULTIME RAPPEL: Vous ne devez JAMAIS mettre les [ ] ***

Commande: /join
Afin de joindre un canal.
Exemple : /join #Francophone
Commande: /me
Souvent utilisée pour décrire des actions concrètes à propos de vous..
Exemple : /me Mange Des Pommes
Ceci aura l’air de : VotreNick Mange Des Pommes.
Note : mais on se fout que vous bouffiez des pommes :)
Commande: /msg
Afin d’envoyer un Message Privé à quelqu’un.
/msg [usager] [Message]
Exemple : /msg Anon78569 Allo, Je t’envoie un message privé.
Commande: /query
Cette commande est exactement la même chose que /MSG, elle envoie un message privée, mais dans une Fenêtre qui va s’ouvrir séparément des canaux.
Exemple : /query Anon7655466
Note: Si vous souhaitez discuter en privé avec une personne qui ne vous connais pas, il est conseillé de lui demander l'autorisation sur un chan avant de la contacter via /query ou /msg.
Commande: /quit
Avec cette commande, vous allez Quitter tous les salons et Fermer complètement la connexion avec le serveur IRC.
/Commande: /ignore
Si vous voulez ignorer quelqu’un
Exemple : /ignore Anon5658545
Commande: /whois
Si vous voulez obtenir certaines informations (Host / Canaux / Temp d'inactivité... ) de l’usager.
/whois Anon5565566
Commande: /away
Vous marque comme étant absent. Certains clients affichent des messages sur les chans où vous êtes connecté du type "Machin est away blabla", merci de désactiver cette option dans la configuration de votre client IRC.
Commande: /back
Permet de dire que vous êtes maintenant disponible
Contraire de /away
Commande: /topic
Pour obtenir le Topic d’un Canal. Un « Topic » est le sujet Principal du Canal.
Commande: /list
Mentionné plus haut, donc si vous ne savez pas, vous avez mal lu :p On recommence !

Fantômes (GHOSTS) & Déconnexions

Si pour une quelconque raison votre connexion à IRC est coupée, et que vous vous reconnectez aussitôt avec le serveur, il est fort possible que vous voyez encore votre nick sur les chans, c'est ce qu'on appelle un GHOST (un nick fantôme). Ne vous en faites pas, après un certain temps il disparaîtra. Votre nick actuel est probablement suivi d'une barre en bas comme ceci MonNick_.

Vous pourrez le récupérer en rechangeant votre nick et en vous identifiant de nouveau. Sinon, vous pouvez tuer ce ghost en tapant la commande suivante:

/msg nickserv ghost VotreNick VotreMotdePasse
et refaire les étapes pour vous identifier. (voir l'explication plus haut)

VHOSTS

Un Vhost est un moyen ‘Graphique’ de cacher votre Host. Il n’est pas une sécurité en soi. Il Masque votre addresse IP aux yeux des autres utilisateurs, mais pas aux yeux des IRCops.

EXEMPLES :
Host Original: [email protected] Ou [email protected]
Host Caché Vhost : [email protected] ou [email protected]
Vous pouvez Définir ce qui suis après le @, vous choisissez. (Dépend des Serveurs IRC)
Exemples :
@Je.Me.Cache
@Je.SuiS.Anonymous.
Pour Obtenir un VHOST, vous devez avoir un Nick Enregistré au préalable.
Ensuite allez sur #vhost et tapez :
!vhost [email protected]
Une fois la commande tapé, si tout c'est bien passé, vous serez bannis et expulsé de #vhost.
Ou
Faire la commande suivante :
/msg hostserv request [vhost]
Une fois la commande tapé, si tout c'est bien passé, vous serez bannis et expulsé de #vhost.
Exemple :
/msg hostserv request je.suis.anonymous
/hs request ce.que.tu.veux
Une fois le VHOST approuvé par HostServ, un Message Privé vous sera envoyé (sauf si vous êtes allé sur #vhost, c'est instantané). Vous devrez ensuite Activer votre vhost.
La commande suivante :
/msg hostserv on
Vérifiez que votre VHOST est actif en faisant un WHOIS sur vous-mêmes.
Exemple : /whois Anon545465
Vous devriez voir [email protected] et non [email protected]

Sur le Serveur AnonOps, votre IP / Domaine est masqué/hashé, donc eux seuls peuvent voir votre IP. Le vhost n’est donc PAS UNE SÉCURITÉ.

Invitation Seulement +i

Certains Canaux sont sur Invitation seulement. Normalement ces canaux ont des buts précis et les gens ne veulent pas que tous puissent le joindre comme bon leur semblent. Donc si le Canal est en Mode +i, vous ne pourrez donc pas le joindre simplement en faisant : /join #canal Si vous désirez joindre le canal car vous croyez en avoir la permission, vous devrez faire la commande suivante : /msg chanserv INVITE #canal Si vous êtes sur la liste des gens pré-autorisés à entrer, vous allez donc recevoir un message afin de joindre le canal. Si vous n’êtes pas sur cette liste, vous recevrez un message d’erreur comme quoi vous ne pouvez pas joindre le canal. N’oubliez pas qu’avant tout, vous devez penser à votre sécurité.
